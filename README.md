## #️⃣ 대회 소개

#### [주제]
- 제주도 특산물의 가격을 예측하는 AI 모델 개발 및 인사이트 발굴

#### [주최 / 주관]
- 주최: 제주특별자치도
- 주관: 제주테크노파크, 데이콘
  
#### [규모]
- 2,000 여명 참가

#### [규칙]
- 심사 기준: RMSE 
- Public Score: 2023.03.04~2023.03.17의 데이터로 측정 (2주)
- Private Score: 2023.03.04~2023.03.31의 데이터로 측정 (4주 대회 마감 후 공개)
